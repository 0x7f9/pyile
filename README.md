# Pyile

A real time Python monitoring tool for tracking changes on Windows machines with detailed logs whenever a change is detected.
```bash
# Install needed packages
python pip install -r requirements.txt
```
**Features**

- Monitor single or multiple file directories in real time (Added, Modified, and Deleted)
- Create local backups of individual files or entire directories, detected changes will trigger futher backups
- Duplicate file dection Using hashing
- Option to scan all current files in selected directories for duplicates
- Windows desktop notification support 
- Built in tracking features and detailed logging suppport
- Configuration files keeping track of selected directories or options making the next monitoring session easier
